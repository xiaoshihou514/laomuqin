### 2. ASR INITIALIZATION & TRANSCRIPTION

**File: services/speech.dart (50 lines)**

OVERVIEW:
Handles both ASR transcription and ZDPP punctuation using sherpa-onnx C++ bindings

TRANSCRIPTION WORKFLOW:

```dart
static Future<String> fromWAV(
  so.OnlineModelConfig? modelConfig,
  so.OfflinePunctuationConfig? zdppConfig,
  Float32List data,
  int sampleRate,
) async {
  if (modelConfig == null) return "";
  
  // 1. Initialize Dart-C++ bindings
  so.initBindings();
  
  // 2. Create recognizer with model config
  final asrModel = so.OnlineRecognizer(
    so.OnlineRecognizerConfig(model: modelConfig)
  );
  
  // 3. Create stream for audio processing
  final stream = asrModel.createStream();
  
  // 4. Feed audio data (PCM samples and sample rate)
  stream.acceptWaveform(samples: data, sampleRate: sampleRate);
  
  // 5. Process stream while ready (streaming ASR)
  while (asrModel.isReady(stream)) {
    asrModel.decode(stream);
  }
  
  // 6. Get final result
  final res = asrModel.getResult(stream);
  final raw = res.text;
  
  // 7. Cleanup (release C++ objects)
  asrModel.free();
  stream.free();
  
  // 8. Optional: Apply punctuation correction
  if (zdppConfig == null || raw.isEmpty) {
    return raw;
  } else {
    return zdpp(raw, zdppConfig);
  }
}
```

KEY PARAMETERS:
- Input: Float32List (f32le PCM audio)
- Sample rate: Passed as parameter
- Format: Mono audio (channels implied by OnlineModelConfig)
- Output: Text string

PUNCTUATION HANDLING:

```dart
static zdpp(String input, so.OfflinePunctuationConfig zdppConfig) {
  so.initBindings();
  final zdppModel = so.OfflinePunctuation(config: zdppConfig);
  final result = zdppModel.addPunct(input);
  zdppModel.free();
  return result;
}
```

### 2. ASR SERVICE WITH BACKGROUND PROCESSING

**File: data/services/asr_service.dart (67 lines)**

MAJOR IMPROVEMENT: Background Isolate Processing

```dart
class AsrService {
  /// Transcribe audio in background isolate (non-blocking UI)
  static Future<String> transcribe({
    required String settingJson,
    required Float32List samples,
    required int sampleRate,
  }) {
    return compute(_isolateEntry, _AsrParams(
      settingJson: settingJson,
      samples: samples,
      sampleRate: sampleRate,
      token: ServicesBinding.rootIsolateToken!,  // Pass isolate token
    ));
  }
  
  /// Entry point for background isolate
  static Future<String> _isolateEntry(_AsrParams params) async {
    // Initialize bindings in background isolate
    BackgroundIsolateBinaryMessenger.ensureInitialized(params.token);
    
    // Deserialize settings
    final setting = AsrSetting.fromJson(params.settingJson);
    if (!setting.isValid) return '';
    
    // Initialize sherpa-onnx
    so.initBindings();
    
    // Create recognizer
    final recognizer = so.OnlineRecognizer(
      so.OnlineRecognizerConfig(model: setting.model),
    );
    final stream = recognizer.createStream();
    
    // Process audio
    stream.acceptWaveform(samples: params.samples, sampleRate: params.sampleRate);
    while (recognizer.isReady(stream)) {
      recognizer.decode(stream);
    }
    
    // Get result and cleanup
    final text = recognizer.getResult(stream).text;
    recognizer.free();
    stream.free();
    
    return text;
  }
}

class _AsrParams {
  final String settingJson;
  final Float32List samples;
  final int sampleRate;
  final RootIsolateToken token;
}
```

KEY BENEFITS:
1. UI remains responsive during transcription
2. Heavy C++ binding work off main thread
3. Result returned via Future
4. Error handling via try-catch in caller

```dart
class SharedPreferencesService {

  Future<void> setAsrEnabled(bool value) async {
    final prefs = await SharedPreferences.getInstance();

    return prefs.getString(_kAsrModelSettings);

    }
  }
}
```

### 4. COMMAND PATTERN FOR ASYNC OPERATIONS

**File: utils/command.dart**

PATTERN: Result-based Error Handling

```dart
// Base class for 0-argument async operations
class Command0<T> {
  Command0(this._function);
  
  final Future<Result<T>> Function() _function;
  
  Future<Result<T>> execute() => _function();
}

// Base class for 1-argument async operations
class Command1<U, T> {
  Command1(this._function);
  
  final Future<Result<U, T>> Function(U) _function;
  
  Future<Result<T>> execute(U arg) => _function(arg);
}

// Result type: Either Ok<T> or Error<E>
sealed class Result<T> {
  const Result();
}

class Ok<T> extends Result<T> {
  const Ok(this.value);
  final T value;
}

class Error<T> extends Result<T> {
  const Error(this.error);
  final Exception error;
}
```

USAGE IN VIEWMODEL:

```dart
    load = Command0(_load)..execute();
    toggleAsr = Command0(_toggleAsr);
    saveAsrModelSettings = Command1<String?, void>(_saveAsrModelSettings);
  }
  
  Future<Result<void>> _load() async {
    final asrResult = await _settingsRepository.isAsrEnabled();
    if (asrResult is Ok<bool>) {
      _asrEnabled = asrResult.value;
    }
    notifyListeners();
    return Result.ok(null);
  }
  
  Future<Result<void>> _toggleAsr() async {
    _asrEnabled = !_asrEnabled;
    notifyListeners();
    return _settingsRepository.setAsrEnabled(_asrEnabled);
  }
  
  Future<Result<void>> _saveAsrModelSettings(String? json) async {
    final result = await _settingsRepository.setAsrModelSettings(json);
    if (result is Ok) {
      _asrModelSettingsJson = json;
      notifyListeners();
    }
    return result;
  }
}
```

BENEFITS:
1. Explicit error handling
2. No exceptions thrown, predictable results
3. Type-safe (Ok vs Error known at compile time)
4. Chainable operations (in extension methods)

```dart
    _startTime = DateTime.now();
    _timer = Timer.periodic(const Duration(milliseconds: 100), (_) {

          _elapsed = DateTime.now().difference(_startTime);
        });
      }
    });
```

```dart
Future<void> _done() async {
  if (!_recording || _processing) return;
  
  _timer?.cancel();
  _cancelled = true;
  _recorder.stopStreamingData();
  _recorder.deinit();
  setState(() {
    _recording = false;
    _processing = true;  // Show spinner
  });
  
  // Convert buffer to Float32List
  final samples = Float32List.fromList(_bytes);
  
  String transcript = '';
  try {
    // Transcribe in background isolate
    transcript = await AsrService.transcribe(
      settingJson: widget.settingJson,
      samples: samples,
      sampleRate: _kSampleRate,  // 22050 Hz
    );
  } catch (_) {
    transcript = '';
  }
  
  // Return to caller
  if (mounted) Navigator.pop(context, transcript);
}
```

```dart
    decoder: p.needsDecoder ? _decoderCtrl.text.trim() : null,
    joiner: p.needsJoiner ? _joinerCtrl.text.trim() : null,
```
